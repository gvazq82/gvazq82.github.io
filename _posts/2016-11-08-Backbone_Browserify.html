Migrate AMD RequireJS to Browserify JS

Backbone JS, Browserify, Jasmine JS

Problem

Migrate existent RequireJS application to Browserify including npm as package manager.

Objective

Migrate existent AMD application to Browserify.

Migration Process

  • Create a new package.json at the root folder of my new application.
  • Install the required dependencies for my project: - backbone - underscore - jQuery and the following dev dependencies: - bootstrap-sass - browserify - jasmine - jstify
  • Replace all the RequireJS packages defined as: define(['backbone'], function(Backbone) { ... return {}; }); By CommonJS packages: var Backbone = require('backbone'); ... module.exports = {};
  • Create a main.js file responsible to include all dependencies and some setup required to work with Backbone/JQuery. var Backbone = require('backbone'); var $ = require('jquery'); Backbone.$ = $; As well require to execute the App.Start() function responsible to initialize the routers.
  • Include the require scripts to build my application inside package.json.
